# 🏠 Housing Price Prediction using Linear Regression

## 📌 Project Overview

This project focuses on predicting house prices using **Multiple Linear Regression** based on various property features such as area, number of bedrooms, bathrooms, stories, parking availability, and furnishing status.

The objective is to analyze how different housing features influence property prices and develop an accurate predictive model.

---

## 🎯 Problem Statement

To build a regression model that predicts house prices based on multiple property-related features and evaluate its performance using appropriate statistical metrics.

---

## 🛠 Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

---

## 🔄 Project Workflow

### 1️⃣ Data Preprocessing

- Checked for missing values  
- Removed duplicate records  
- Converted categorical variables using **One-Hot Encoding**  
- Split the dataset into training and testing sets  

---

### 2️⃣ Model Building

- Applied **Multiple Linear Regression**  
- Trained the model on the training dataset  

---

### 3️⃣ Model Evaluation

- Calculated **R² Score**  
- Computed **Mean Squared Error (MSE)**  
- Computed **Root Mean Squared Error (RMSE)**  
- Compared **Actual vs Predicted values** using visualization  

---

## 📊 Model Evaluation Metrics

- **R² Score** – Measures how well the model explains the variance in house prices  
- **RMSE** – Measures the average prediction error in original price units  
- Residual analysis was performed to validate regression assumptions  

---

## 📈 Visualizations

- Correlation Heatmap  
- Actual vs Predicted Prices Scatter Plot  
- Residual Plot  

These visualizations help in understanding feature relationships and evaluating model performance.
